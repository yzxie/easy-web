# 服务统一管理平台
# 功能
1. 作为easy-ui的后端，为easy-log日志分析服务，easy-monitor服务器性能分析服务做数据的统一展示平台；
2. 采用react框架搭建，基于单页面应用的设计，每个服务对应一个页面。
# 技术栈
1. spring boot
2. spring shiro
3. dubbo
4. websocket
5. mybatis
